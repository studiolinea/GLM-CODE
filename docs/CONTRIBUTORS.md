## Comment devenir contributeur

1. **Fork le projet** : Créez un fork du dépôt GitHub
2. **Créez une branche** : Pour votre fonctionnalité ou correction
3. **Développez** : Suivez les bonnes pratiques de codage
4. **Testez** : Assurez-vous que tous les tests passent
5. **Créez une PR** : Décrivez vos changements en détail
6. **Répondez aux commentaires** : Soyez réactif aux reviews

### Bonnes pratiques

- Utilisez des commits atomiques
- Documentez votre code
- Écrivez des tests
- Suivez le style de code existant
- Soez respectueux et constructif
